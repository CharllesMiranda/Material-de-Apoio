📌 Cores e Fundos (Background)
🔹 Propriedade color
  • O que é: Define a cor do texto em um elemento.

🔹 Formas de definir cores
  • Nomes de cores: Como blue, red, green. São limitados, mas fáceis de lembrar.
  • Hexadecimal: É o formato mais usado. Ex.: #0000FF (azul). Ele usa 6 dígitos (ou 3 dígitos abreviados, #00F). Os dois primeiros dígitos são para vermelho, os dois do meio para verde e os dois últimos para azul (padrão RGB).
  • RGB: Define as cores em valores de 0 até 255 para cada canal (vermelho, verde e azul). Ex.: rgb(0,0,255) (azul).
  • RGBA: Igual ao RGB, mas com transparência. O último número vai de 0 até 1 (0 é totalmente transparente e 1 é totalmente opaco). Ex.: rgba(0,0,255,0.5) (azul com 50% de transparência).
  • HSL: Significa Hue (matiz), Saturation (saturação) e Lightness (luminosidade). Ex.: hsl(240,100%,50%) (azul puro). É ótimo para ajustar tonalidades de forma mais intuitiva.
  • HSLA: Igual ao HSL, mas com transparência no quarto valor. Ex.: hsla(240,100%,50%,0.5).

📌 Unidades de Medida (CSS)
🔹 Absolutas / Relativas
  • O que é: São as unidades usadas para definir tamanhos no CSS, como largura, altura, fontes, margens e espaçamentos.
  • Quando usar: Sempre que precisar definir tamanhos de forma fixa (absoluta) ou adaptável (relativa) no seu layout.
  • Exemplo: px, %, em, rem, vh, vw, etc.

💻 Exemplos de Unidades

  • px (pixels): Unidade absoluta mais comum. Ex.: font-size: 16px;
  • % (porcentagem): Relativa ao elemento pai. Ex.: width: 50%;
  • em: Relativa ao tamanho da fonte do elemento pai. Ex.: padding: 2em;
  • rem: Relativa ao tamanho da fonte raiz (html). Ex.: margin: 1.5rem;
  • vh (viewport height): Relativa à altura da janela do navegador. Ex.: height: 50vh;
  • vw (viewport width): Relativa à largura da janela do navegador. Ex.: width: 80vw;
  • vmin/vmax: Relativas ao menor ou maior valor entre largura e altura do viewport.
  • cm / mm / in: Unidades físicas (centímetro, milímetro, polegada) — raramente usadas na tela.

📌 Propriedades de Display e Layout
🔹 Display padrão: block / inline / inline-block / none
  • O que é: A propriedade display define como um elemento é exibido na página.
  • Quando usar: Sempre que precisar controlar se o elemento ocupa toda a linha, se fica ao lado de outros ou se deve desaparecer.
  • Exemplo:
    • block → O elemento ocupa toda a largura disponível (ex.: <div>, <p>).
    • inline → O elemento ocupa apenas o espaço do conteúdo e fica na mesma linha (ex.: <span>, <a>).
    • inline-block → Mistura os dois: fica na mesma linha, mas permite definir largura/altura.
    • none → O elemento não é exibido na tela.

🔹 Padding
  • O que é: A propriedade padding em CSS serve para criar espaçamento interno — ou seja, o espaço entre o conteúdo do elemento e a sua borda.
  • Quando usar: Você usa sempre que quiser dar respiro ao conteúdo interno do elemento.
  • Pensa assim:
    • margin = espaço fora do elemento.
    • padding = espaço dentro do elemento.
  • Exemplo:
    • Aumentar o espaço entre o texto e a borda de um botão.
    • Criar espaço interno em cards, seções, caixas de texto.
    • Evitar que o conteúdo “grude” nas bordas.

  • Quando usar: Quando quiser alterar a forma ou posição de um elemento visualmente (sem afetar o layout ao redor).

🔹 Position
  • Static: Padrão. O elemento segue o fluxo normal da página.
  • Quando usar: Quando não precisa de posicionamento especial.
  • Relative: Permite mover o elemento em relação à sua posição original.
  • Quando usar: Quando quer ajustar algo levemente sem tirá-lo do fluxo.
  • Absolute: Posiciona em relação ao ancestral mais próximo com posição definida (ou ao body).
  • Quando usar: Quando quer um item fixo dentro de uma área específica.
  • fixed: Fica fixo na tela, mesmo rolando a página
  • Quando usar: Ideal para menus fixos, botões flutuantes, etc.
  • Sticky: Fica normal até atingir um ponto e depois “gruda” na tela.
  • Quando usar: Usado para cabeçalhos ou menus que grudam ao rolar.

🔹 Z-index
  • O que é: A propriedade z-index define a ordem de empilhamento (camadas) dos elementos que se sobrepõem na tela.
  • Quando usar: Quando quiser definir quem aparece na frente e quem fica atrás.
  • Como funciona:
  • Quanto maior o valor do z-index, mais à frente o elemento aparece.
  • Quanto menor, mais ao fundo.
  • Só funciona em elementos que têm um position diferente de static (relative, absolute, fixed, ou sticky).

🔹 Pseudo-classes e Pseudo-elementos
  • Pseudo-classes (:) Servem para definir um estilo em um estado específico de um elemento, como quando o mouse passa em cima ou o campo está em foco.
  • Pseudo-classe
    • :hover
      • O que faz: Ativado quando o mouse passa sobre o elemento.
      • Quando usar: Criar efeitos de destaque ou animações.
    • :focus
      • O que faz: Quando o elemento (geralmente um input) recebe foco.
      • Quando usar: Personalizar campos de formulário quando estão selecionados.
    • :active
      • O que faz: Quando o elemento está sendo clicado.
      • Quando usar: Dar feedback visual em botões ou links.
  • Pseudo-elementos (::) Usados para criar partes “falsas” dentro de um elemento, sem precisar adicionar tags extras no HTML.
  • Pseudo-elemento
    • ::before
      • O que faz: Adiciona conteúdo antes do elemento.
      • Quando usar: Criar detalhes visuais, ícones, decorações.
    • ::after
      • O que faz: Adiciona conteúdo depois do elemento.
      • Quando usar: Criar setas, enfeites, efeitos extras.
